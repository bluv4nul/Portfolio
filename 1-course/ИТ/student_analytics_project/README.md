# Анализ успеваемости студентов

## Описание проекта

Данный проект представляет собой консольное приложение для анализа успеваемости студентов, разработанное на языке C. Программа позволяет загружать данные студентов из CSV-файлов, рассчитывать статистические показатели с использованием библиотеки GSL, сортировать студентов по различным критериям и сохранять результаты в виде отчетов.

Проект демонстрирует:
- Иерархическую структуру каталогов
- Подробную документацию в стиле Google
- Использование внешних библиотек (GSL, libcurl)
- Сборочные файлы для GNU Make и Meson

## Структура проекта

```
student_analytics/
├── bin/                  # Директория для исполняемых файлов
├── build/                # Директория для объектных файлов
├── data/                 # Директория для данных
├── include/              # Заголовочные файлы
│   └── student.h         # Определение структуры Student и прототипы функций
├── lib/                  # Директория для библиотек
├── src/                  # Исходные файлы
│   ├── main.c            # Главный модуль программы
│   └── student.c         # Реализация функций для работы с данными студентов
├── Makefile              # Сборочный файл для GNU Make
├── meson.build           # Сборочный файл для Meson
└── README.md             # Документация проекта
```

## Зависимости

Для сборки и запуска проекта требуются следующие зависимости:
- GCC (компилятор C)
- GNU Make или Meson (системы сборки)
- GSL (GNU Scientific Library)
- libcurl (библиотека для работы с HTTP)

### Зависимости в Ubuntu или Windows WSL2 Ubuntu

Выполните в терминале:

```bash
sudo apt update
sudo apt install -y libgsl-dev libcurl4-openssl-dev
```

## Сборка проекта

### С использованием GNU Make

```bash
# Сборка проекта
make

# Очистка проекта
make clean

# Полная очистка проекта
make distclean

# Запуск с тестовыми данными
make run

# Проверка зависимостей
make check-deps

# Справка
make help
```

### С использованием Meson

```bash
# Создание директории сборки
meson setup build

# Компиляция проекта
meson compile -C build

# Запуск тестов
meson test -C build

# Очистка
rm -rf build
```

## Использование программы

```bash
# Загрузка данных студентов из CSV-файла
./bin/student_analytics load <файл.csv>

# Расчет статистики и вывод результатов
./bin/student_analytics stats <файл.csv>

# Сортировка студентов и сохранение отчета
./bin/student_analytics sort <файл.csv> <критерий> <отчет>

# Загрузка дополнительных данных и сохранение отчета
./bin/student_analytics fetch <файл.csv> <url> <отчет>

# Генерация тестовых данных
./bin/student_analytics generate <файл.csv> <количество>
```

Критерии сортировки: `average` (средний балл), `median` (медиана), `rating` (рейтинг).

## В чём ценность проекта

Проект разработан для студентов 1-го курса ИВТ (информатика и вычислительная техника) и демонстрирует:
- Более блиский к профессиональному подход к организации кода и документации
- Работу с внешними библиотеками
- Создание и использование сборочных систем
- Обработку данных и базовый/простой статистический анализ
